# 3LM (3 Lazy Monks)

A command line tool for generating a wiki page for Github repositories, powered by GPT-4 API.


## Demo
![](https://iustino.com/assets/img/3lm_demo.gif)

## Install

### For Development
Run ```pip install -r requirements.txt```

(Use brew install)
